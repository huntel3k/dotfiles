## Requirements

Ensure you have following installed on the system:

### Git 

```
pacman -S git
```

### Stow

```
pacman -S stow
```

## Installation

First check out the dotfiles repo in your $HOME directory using git

```
$ git clone git@github.com/huntel3k/dotfiles.git
$ cd dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
